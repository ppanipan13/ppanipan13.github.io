1. Create a free GitHub account if you do not already have one.

3. Upload everything in this folder to the repository root.
4. GitHub Pages will publish it at `https://ppanipan13.github.io/`.
5. Submit `https://ppanipan13.github.io/sitemap.xml` in Google Search Console.
