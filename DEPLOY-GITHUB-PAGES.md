# Free Website Deployment: GitHub Pages

Your site is ready for this free URL:

https://ppanipan13.github.io/

## Steps

1. Go to https://github.com/new
2. Create a public repository named exactly:

   `ppanipan13.github.io`

3. Upload all files from this folder into the repository root.
4. Wait about 1-3 minutes.
5. Visit https://ppanipan13.github.io/

## Files To Upload

- `index.html`
- `panipan-pansang-hero.png`
- `digital-journal-panipan-pansang.jpg`
- `robots.txt`
- `sitemap.xml`
- `.nojekyll`
- `README.md`
- `profile-copy.md`
- `assets/documents/panipan-pansang-cv.pdf`
- `assets/certificates/` and every PDF inside it
